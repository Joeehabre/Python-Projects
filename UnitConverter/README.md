# 📏 Unit Converter

Convert between length, weight, temperature, speed, and volume units — with a convert-again loop and clean output.

---

## ✨ Features

- 🔁 **Loop** — convert multiple times without restarting
- 5 categories with extensive unit support
- Clean `:.6g` output (no trailing zeros)
- Input validation shows valid unit list on bad input

---

## 📐 Supported Units

| Category | Units |
|---|---|
| Length | `mm` `cm` `m` `km` `inch` `ft` `yd` `mile` |
| Weight | `mg` `g` `kg` `tonne` `oz` `lb` |
| Temperature | `c` (Celsius) `f` (Fahrenheit) `k` (Kelvin) |
| Speed | `m/s` `km/h` `mph` `knot` `ft/s` |
| Volume | `ml` `l` `m3` `tsp` `tbsp` `fl_oz` `cup` `pint` `gallon` |

---

## 🚀 Run

```bash
python UnitConv.py
```

No dependencies — stdlib only.
